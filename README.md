# ALKIF

Abundant Love Kids Foundation (ALKIF) website template and documentation.

## Structure
- `ALKIF - Template/` contains the website pages, assets, and PHP helpers.
- `ALKIF - Documentation/` contains the template documentation.
- `pages.md` lists available pages.

## Usage
- Open `ALKIF - Template/index.html` in a browser for a static preview.
- If you use PHP features, serve the folder with a local PHP server.
